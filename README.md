# Rxss-Scan

**Rxss-Scan** 是一款轻量级的 Burp Suite 插件，用于自动化检测流量中的反射型 XSS 漏洞。

### 🌟 功能特点
* **全自动检测**：只需开启 Burp Proxy 正常浏览网页，插件会自动在后台扫描。
* **智能去重**：同一个接口（Path + Params）只扫一次，不重复发包，不干扰工作。
* **低感运行**：异步多线程设计，不卡顿，不影响网页加载速度。
* **独立面板**：提供专门的 Tab 页面查看结果，支持请求/响应联动显示。

### 🔍 扫描逻辑
1. 监控 Proxy 流量中的 `GET` 请求。
2. 识别 `Content-Type` 为 `text/html` 的页面。
3. 自动在 URL 参数中注入探测 Payload：`\"<>abc`。
4. 检查响应正文中是否原样反射了该内容。
